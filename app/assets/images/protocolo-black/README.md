# Imagens — Protocolo Black

Coloque aqui os arquivos de branding da mentoria.

## Arquivos esperados

| Arquivo | Uso | Tamanho sugerido |
|---|---|---|
| `logo.png` | Logotipo exibido no header da landing page | 200x200px (quadrado ou com padding transparente) |
| `banner.jpg` | Banner principal da hero section | 1920x1080px (ou 1600x900) |

## Como adicionar

1. Coloque os arquivos nesta pasta com exatamente os nomes acima (`logo.png` e `banner.jpg`).
2. Commit e push para o branch `main`.
3. A GitHub Actions (ou o deploy manual) vai rebuildar o Flutter web automaticamente.

## Formatos aceitos

- Logo: **PNG com fundo transparente** (preferencial) ou SVG.
- Banner: JPG ou PNG. Fundo escuro funciona melhor com o design dark do app.
